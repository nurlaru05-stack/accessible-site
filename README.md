# DataHub Университета

Интерактивный сайт для представления университета, доступный для людей с ограничениями по зрению и слуху.  

## Особенности
- Адаптивный дизайн
- Плавные анимации
- Контрастный режим и увеличение шрифта
- Использование JSON/API для загрузки данных
- 3D-тур интеграция

## Структура
- `index.html` — главная страница
- `university.html` — информация об университете
- `programs.html` — академические программы
- `admission.html` — приём
- `tour.html` — виртуальный 3D-тур
- `international.html` — международное сотрудничество
- `compare.html` — сравнение программ
- `css/style.css` — стили
- `js/script.js` — скрипты
- `data/` — JSON файлы с данными

## Запуск
Откройте `index.html` в браузере. Для работы с API используйте локальный сервер (например, Live Server в VS Code).
